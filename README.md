## Configurações
### Projeto em Angular
**Node Version:** v14.20.1
### Comandos do CLI
1. Verificar as versões do Node instaladas:
   nvm list

2. Usar a versão específica do Node:
   nvm use v14.20.1

3. Checar a versão em uso:
   nvm list

4. Instalar dependências:
   npm install

5. Compilar a aplicação:
   npm start
### Acessar no Navegador
  http://localhost:4200/
